# MANUAL DE USUARIO - Juego Memes 8M

## 🎭 Introducción

Bienvenido al **Juego Memes 8M**, un proyecto educativo desarrollado para concienciar sobre los prejuicios y bulos relacionados con el Día Internacional de la Mujer Trabajadora (8M). El objetivo es identificar la realidad verdadera detrás de memes populares que perpetúan estereotipos negativos.

En este juego, se te presentarán memes junto con varias opciones de "realidad". Tu tarea es elegir la opción correcta para ganar puntos y contribuir a la desmitificación de estos contenidos.

---

## 📋 Requisitos del Sistema

Antes de comenzar, asegúrate de tener instalado:

- **Java JDK 25** o superior (verifica con `java --version`)
- Los archivos de datos en la carpeta `datos/` deben estar presentes
- Los archivos JAR de dependencias en `lib/` (incluyendo JSON y JUnit)

**Nota:** Si no tienes Java instalado, descárgalo desde [oracle.com](https://www.oracle.com/java/technologies/javase-downloads.html) o usa OpenJDK.

## 🚀 Instalación y Configuración

1. **Descarga el proyecto:**
   - Clona el repositorio o descarga el ZIP desde GitHub.

2. **Verifica los archivos:**
   - Asegúrate de que `datos/memes.txt`, `datos/realidades.json` y `datos/soluciones.xml` existan.
   - Los JARs en `lib/` deben estar presentes.

3. **Compila el proyecto:**
   - Abre una terminal en la raíz del proyecto.
   - Ejecuta: `javac -cp "lib/json-20251224.jar" -d compilados fuentes/*.java`

4. **Ejecuta el juego:**
   - Desde la terminal: `java -cp "compilados;lib/json-20251224.jar" JuegoMemes`

**Nota para Linux/Mac:** Usa `:` en lugar de `;` en los classpath.

---

## 🎮 Cómo Jugar

1. **Inicio del juego:**
   - El programa verifica automáticamente que todos los archivos necesarios estén presentes.
   - Si falta algún archivo, el juego se detendrá con un mensaje de error.

2. **Selección de rondas:**
   - Elige el número de rondas (entre 5 y 10). Cada ronda representa un meme diferente.

3. **Juego principal:**
   - Se muestra un meme en pantalla.
   - Aparecen varias opciones de "realidad" (una correcta, otras falsas).
   - Elige la opción que crees que es la realidad verdadera (introduce el número correspondiente).

4. **Puntuación:**
   - Por cada respuesta correcta: +10 puntos.
   - Por cada respuesta incorrecta: 0 puntos.
   - Se muestra el marcador después de cada ronda.

5. **Fin del juego:**
   - Al completar todas las rondas, se muestra tu puntuación final.
   - Si entras en el top 3, introduce tu nombre para guardar la puntuación.
   - Se muestran las mejores puntuaciones guardadas.

6. **Salida:**
   - El juego se despide y puedes cerrar la aplicación.

## 🛠️ Solución de Problemas

### Problema: "Java no reconocido como comando"
- **Solución:** Asegúrate de que Java esté en el PATH. Instala JDK y configura las variables de entorno.

### Problema: "Archivo no encontrado"
- **Solución:** Verifica que los archivos en `datos/` estén presentes y no corruptos.

### Problema: "Error al compilar"
- **Solución:** Comprueba que el JDK 25 esté instalado y que las rutas a los JARs sean correctas.

### Problema: "El juego no inicia"
- **Solución:** Asegúrate de compilar primero. Si usas IDE, configura el classpath correctamente.

### Problema: "Tests fallan"
- **Solución:** Los tests requieren JUnit. Ejecuta desde terminal con los comandos en Herramientas.txt.
